Общие положения

Программа MKStudio предназначена для проведения следующих работ с микропроцессорными контроллерами (МК) из состава ПТК УМИКОН:

  • настройки параметров,
  • метрологических поверок,
  • измерений сигналов и управлений в ручном режиме для наладки оборудования,
  • исследований входных аналоговых сигналов,
  • создания собственных видов представления информации и элементов управления,
  • изменения предустановленных конфигураций (проектов) для различных МК и создания собственных,
  • помощи при отладке программного обеспечения (ПО) МК.

MKStudio взаимодействует с МК посредством конвертеров интерфейсов USB-RS485/DCL или RS232-RS485/DCL. Используется расширение протокола MODBUS RTU (MODBUS MIKKON). Для интерфейсов при старте программы задаются их настоечные параметры. В MKStudio указывается список МК (адреса и наименования), с которыми осуществляется взаимодействие. MKStudio позволяет отображать и записывать параметры из информационной базы МК в режиме реального времени. Отображение организуется в виде таблицы в центральной области главного окна программы.
