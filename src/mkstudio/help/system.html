Требования к оборудованию:

  • ПЭВМ класса IPM PC с интерфейсами RS232 или USB.
  • Наличие совместимого конвертера интерфейсов RS232-RS485 или USB-RS485 (для МК с интерфейсом RS485).
  • ОС Windows 98/Me/2000/XP/2003, русифицированные версии.
  • Не менее 20МБайт дискового пространства для успешной установки.
  • Права Администратора для установки.

При подключении МК через конвертер интерфейсов RS232-RS485 при использовании ОС Windows 98/ME необходим конвертер интерфейсов RS232-RS485 с автоматическим определением направления передачи.

Подключение оборудования:

  • Присоединить к соответствующему разъему ПЭВМ конвертер интерфейсов.

    ВНИМАНИЕ! Подключение некоторых конвертеров RS232-RS485 не рекомендуется выполнять при включенном питании ПЭВМ. Для конвертеров из состава ПТК УМИКОН достаточно сначала присоединить конвертер при включенной ПЭВМ, затем включить его питание.

  • Отключить питание модуля.
  • Соединить сеть RS485 на модуле и конвертере, соблюдая полярность.
  • Установить на модуле требуемую скорость и сетевой адрес.
  • Подать питание модуля.

Отключение оборудования:

  • Провести разборку схемы соединений, если требуется; для конвертера RS232-RS485 сначала отключается его питание, потом отсоединяются все разъемы.
