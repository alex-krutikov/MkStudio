Менеджер Микропрограмм Модулей (MMpM) ПТК УМИКОН

Программа MMpM предназначена для управления микропрограммным обеспечением (firmware) модулей ПТК УМИКОН.

Требования к оборудованию:

  • ПЭВМ класса IPM PC с интерфейсами RS232 или USB.
  • Наличие совместимого конвертора интерфейсов RS232-RS485 или USB-RS485.

Условия функционирования:

  • ОС Windows 2000/XP/2003, русифицированные версии.
  • Не менее 10МБайт дискового пространства для успешной установки.
  • Права Администратора для установки.

Последовательность установки программы:

  • Запустить на выполнение файл MMpMSetup.exe, разрешить создание ярлыка на Рабочем столе, дождаться завершения установки.
  • Подключение оборудования:
  • Присоединить к соответствующему разъему ПЭВМ конвертор интерфейсов.

    ВНИМАНИЕ! Подключение некоторых конверторов RS232-RS485 не рекомендуется выполнять при включенном питании ПЭВМ. Для конверторов из состава ПТК УМИКОН достаточно сначала присоединить конвертор при включенной ПЭВМ, затем включить его питание.

  • Отключить питание модуля.
  • Соединить сеть RS485 на модуле и конверторе, соблюдая полярность.
  • Установить требуемую скорость модуля (бит 8 переключателя адреса; в положении "1" - 115200 бод, "0" - 9600 бод).
  • Установить на модуле сетевой адрес 127 (биты 1-7 в положении "1").
  • Подать питание модуля.

Работа с программой:

  1. Запуск программы.
    • Запустить программу с ярлыка на рабочем столе.
    • Выбрать номер порта, к которому присоединен конвертор интерфейсов и скорость обмена.
  2. Чтение и сохранение имеющейся версии firmware.
    • На появившемся формате нажать кнопку "Информация".
    • В окне сообщений появится информация о типе модуля и версии firmware.
    • Нажать кнопку "Прочитать"
    • В ответ на запрос пароля ввести сумму цифр числа запроса.
    • Если пароль введен неверно, повторить действия п.2.
    • Начинается чтение frimware модуля, операция полного чтения и проверки может занять несколько минут.
    • После успешного чтения сохранить firmware в файле с именем, отличным от того файла, что Вы намерены записать.
    • В случае неудачи обмена данными с модулем повторить действия п.2.
  3. Запись новой версии frimware.
    • Нажать кнопку "Записать"
    • Выбрать файл firmware для записи.
    • Если версия firmware предназначена не для этого модуля, запись невозможна, следует обратиться к изготовителю.
    • При соответствии firmware запись разрешена, запрашивается пароль.
    • В ответ на запрос пароля ввести сумму цифр числа запроса.
    • Если пароль введен неверно, повторить действия п.3.
    • Начинается запись frimware модуля, операция полной записи и проверки может занять несколько минут, не следует выключать питание модуля или завершать работу программы в процессе выполнения этих операций.
    • После успешной записи firmware модуль, должен перезапуститься; если этого не произошло см.п.4
    • В случае неудачи обмена данными с модулем повторить действия п.3.
  4. Проверка новой версии frimware.
    • Если Вас не удовлетворяет функционирование новой версии firmware, см.п.5)
    • Если все функционирует нормально, работу MMpM можно завершить
  5. Восстановление firmware
    • Действия аналогичны п.3, но используется ранее сохраненный файл firmware (п.2)

Отключение оборудования:

  • Провести разборку схемы соединений, если требуется; для конвертора RS232-RS485 сначала отключается его питание, потом отсоединяются все разъемы.
  • Не забудьте восстановить скорость и адрес модуля!
